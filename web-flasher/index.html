⚡ ESP32 Network Tester Flasher

Installez le firmware sur votre ESP32-S3 directement depuis le navigateur.

📋 Avant de commencer

1

Utilisez Chrome ou Edge — Firefox ne supporte pas encore l'API Web Serial.

2

Branchez le LilyGo T-Display-S3 en mode USB sur votre PC.

3

Si c'est la première installation, maintenez le bouton BOOT enfoncé pendant la connexion USB pour entrer en mode download.

4

Choisissez le profil hardware ci-dessous, puis cliquez sur Installer le firmware.

🔧 Profil hardware

Choisissez selon votre carte réseau. W5500 SPI est le choix standard.

⚠️ Votre navigateur ne supporte pas l'API Web Serial. Utilisez Chrome ou Edge (version ≥ 89).

📦 Binaires firmware

Les fichiers .bin doivent être compilés avec PlatformIO et placés dans web-flasher/bins/<profil>/ :

bootloader.bin · partitions.bin · boot_app0.bin · firmware.bin

Script de build automatique disponible ci-dessous.

Voir les commandes de build 
# W5500 SPI (profil par défaut)
pio run -e lilygo-t-display-s3
cp .pio/build/lilygo-t-display-s3/bootloader.bin  web-flasher/bins/lilygo-t-display-s3/
cp .pio/build/lilygo-t-display-s3/partitions.bin  web-flasher/bins/lilygo-t-display-s3/
cp ~/.platformio/packages/framework-arduinoespressif32/tools/partitions/boot_app0.bin \
   web-flasher/bins/lilygo-t-display-s3/
cp .pio/build/lilygo-t-display-s3/firmware.bin    web-flasher/bins/lilygo-t-display-s3/

# DP83848 RMII
pio run -e lilygo-t-display-s3-dp83848
cp .pio/build/lilygo-t-display-s3-dp83848/bootloader.bin  web-flasher/bins/lilygo-t-display-s3-dp83848/
cp .pio/build/lilygo-t-display-s3-dp83848/partitions.bin  web-flasher/bins/lilygo-t-display-s3-dp83848/
cp ~/.platformio/packages/framework-arduinoespressif32/tools/partitions/boot_app0.bin \
   web-flasher/bins/lilygo-t-display-s3-dp83848/
cp .pio/build/lilygo-t-display-s3-dp83848/firmware.bin    web-flasher/bins/lilygo-t-display-s3-dp83848/
